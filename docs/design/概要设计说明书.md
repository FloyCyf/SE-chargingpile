# 概要设计说明书 (SDD)

## 1. 系统架构
采用典型的 RESTful API 架构。
- **前端**: 用于展示充电桩状态看板和用户下单页面。
- **后端 (FastAPI)**: 处理调度逻辑、计费逻辑和数据存储。

## 2. 数据库设计 (关键字段)
- **Cars**: id, license_plate, battery_capacity, current_soc
- **Piles**: id, type (Fast/Slow), status (Idle/Charging/Fault)
- **Orders**: id, car_id, pile_id, start_time, total_fee
