# 软件需求规格说明书 (SRS)

## 1. 业务逻辑
系统需根据车辆当前的 SOC (剩余电量) 和选择的充电模式 (快/慢充) 动态计算优先级。

## 2. 功能需求
- **F1 提交请求**: 用户输入车牌、当前电量、目标电量、充电类型。
- **F2 自动调度**: 系统根据优先级函数 P = α(1-SOC) + β(Type) + γ(WaitTime) 实时分配充电桩。
- **F3 模拟充电**: 模拟电量随时间增长，并实时更新费用。
- **F4 欠费/违约处理**: 车辆充满后若不驶离，按分钟扣除占位费。

## 3. 算法参数预设
- α (电量权重): 0.5
- β (类型权重): 0.3 (快充=1, 慢充=0.5)
- γ (等待权重): 0.2
